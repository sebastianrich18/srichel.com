My Projects

I made an AI that plays the game 'flappy bird' better than any human, which you can play with. If you want to see the code, you can find it here. Each bird is powered by a neural network. When the last bird dies, it becomes the parrent of the next generation. It's neural net is changes very slightly and then is passed on to its children. This creates a "survival of the fittest" situation where over time, the birds become more and more tuned. The highest score accieved by one of these birds was 250 thousand, and it never even died. It was flapping away for almost 3 hours and I just turned it off because I was confident it wasn't going to die.
I also made an AI that plays 'snake' but it doesn't work nearly as well as flappy bird did. You can also check out the code for snake here. It uses the same neural net as flappy bird, in the same way. I think that was the biggest reason it doesn't work as well. In flappy bird, every bird is in the same game world. But with snake, each snake needs its own world. This creates a tricky problem of determining which snake should be selected to pass it's altered neural to when a snake dies.
